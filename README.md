# Hamon · 自控系统
 
> **Hamon**（ハモン / 波紋）：在《JoJo的奇妙冒险》中，波纹是一种通过精确的呼吸控制，将微小的生命能量积累放大、转化为可以对抗强大外力的力量的武术。
>
> 这也是这套系统的核心隐喻——用极其微小的、有节律的日常行动，一圈圈向外扩散，积累成足以改变整个生活稳态的力量。
 
---
 
本项目基于知乎作者 **edmond** 的自控理论文章，将其中的两套方法论转化为可日常操作的工具：
 
> 原文：[如何提高自制力？— edmond 的回答](https://www.zhihu.com/question/19888447/answer/1930799480401293785)
 
单文件 HTML，无需安装，无需注册，数据全部存储在本地。支持 PWA，可安装到 iOS / Android 桌面作为独立 App 使用。

## 为什么叫 Hamon
 
波纹（Hamon）这个名字有三层含义：
 
**第一层：隐喻 RSIP 的核心机制**
RSIP（递归稳态迭代协议）的思路是：任何宏观的负面稳态，都可以被一个个微小的"国策"逐步瓦解。就像波纹从中心向外一圈圈扩散——每一个小小的改变，都在边界条件上留下痕迹，最终引发整个系统的相变。
 
**第二层：隐喻 CTDP 的积累机制**
CTDP（链式时延协议）中，每完成一个节点，沉没成本就向外扩散一圈，形成越来越难以打破的屏障。波纹越多，力量越强。
 
**第三层：呼应应用内的波纹预约设计**
应用中的预约功能，正是用 JOJO 风格的波纹动效实现的——按住中心 5 秒，波纹共鸣，承诺生效。Hamon 既是功能的视觉语言，也是整套系统精神的象征。
 
## 这是什么
 
这是一个运行在浏览器里的单文件 Web App，无需安装，无需注册，数据全部存储在本地。
 
它把 edmond 文章中的两套自控方法论转化成了可以日常操作的工具：
 
- **CTDP（链式时延协议）**：通过「神圣座位」「下必为例」「预约机制」三个原理，解决启动困难、中途放弃、破窗效应
- **RSIP（递归稳态迭代协议）**：通过「国策树」逐步改造生活的宏观稳态，用小的局部最优解积累推动整体改变

## 核心原理简介
 
### 为什么我们总是选择刷手机而不是学习
 
文章用一个数学模型解释：人对某个行为的倾向，等于未来价值函数 V(τ) 与权重函数 W(τ) 的积分。W(τ) 的特点是**近端极高、远端趋近于零**——所以即使你知道学习比刷手机重要，近端的即时诱惑还是会赢。这不是意志力问题，是数学结构问题。
 
### CTDP 如何解决这个问题
 
- **神圣座位**：指定一个触发标志，一旦触发就必须以最好状态完成专注。随着节点积累，违规的代价被压缩到 τ=0 处，形成巨大的负向屏障
- **下必为例**：面对任何疑似违规只有两个选项——宣告链条失败清零，或永久允许该行为。没有「这次特殊，下不为例」
- **预约机制（波纹预约）**：把「立刻开始」的高阻力转化为「15分钟后开始」的低阻力，用时间平移绕开 W(τ) 最陡峭的区域
### RSIP 如何改造生活稳态
 
文章借用物理学中的**重整化群理论**：生活在不同时间尺度上由不同因素主导，大量微观的意志力挣扎在宏观尺度上会被平均掉。真正有效的策略，是找到宏观负面稳态的「有效干预节点」，设计针对性的「国策」，通过递归回溯算法自然演化出改变稳态的路径。

## 功能一览
 
| 模块 | 功能 |
|------|------|
| 执行 | 精锐链 / 普通链双链专注计时，按住 2 秒启动，节点可视化 |
| 波纹预约 | JOJO 风格波纹动效，按住 5 秒触发 15 分钟倒计时预约 |
| 判例系统 | 记录「下必为例」的行为边界 |
| 计划 | 本周任务向量坐标系（紧急度×重要度），四象限可视化 |
| 国策树 | RSIP 国策的层级管理，支持崩塌递归、水密隔舱、已熄灭重启 |
| 记录 | 月度统计、操作日志、数据导出导入 |
| 设置 | 专注/休息时长、链条标志、数据管理 |
| 原则 | 系统使用说明 + 理论背景，迷茫时随时查阅 |

## 部署到手机桌面
 
### iOS（iPhone / iPad）
 
1. 注册 [GitHub](https://github.com) 账号（免费）
2. 用 **Safari** 打开：[https://chemit797.github.io/Hamon/](https://chemit797.github.io/Hamon/)
6. 点底部**分享按钮**（方框加箭头）→「添加到主屏幕」→ 确认
> ⚠️ 必须用 Safari，不能用微信内置浏览器或 Chrome
 
### Android
 
用 **Chrome** 打开 [https://chemit797.github.io/Hamon/](https://chemit797.github.io/Hamon/)，点右上角菜单 →「添加到主屏幕」。
